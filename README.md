# AI Resume Analyzer

An intelligent resume analysis tool that uses AI to extract, analyze, and provide insights from resumes in multiple formats.

## Features

- **Multiple Format Support**: Analyze resumes in PDF, DOCX, and image formats
- **AI-Powered Analysis**: Uses advanced NLP and transformers for intelligent analysis
- **Audio Processing**: Support for audio resume submissions
- **Document Extraction**: Extract text and metadata from resumes
- **Interactive Web Interface**: User-friendly Flask-based web application

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

Then open your browser and navigate to `http://localhost:7860`

## Technologies Used

- Flask
- PyTorch
- Transformers
- OpenAI Whisper
- OpenCV
- scikit-learn
- python-docx
- pdfplumber
